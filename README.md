# Simple_Calculator

A clean, user-friendly calculator built with HTML, CSS, and JavaScript. This calculator performs all basic arithmetic operations including addition, subtraction, multiplication, and division. It features a modern dark interface with intuitive button layout and smooth animations.

The calculator supports both mouse and keyboard input, making it convenient for all users. It includes additional functions like percentage calculation, clear, and delete operations. The display shows both current and previous calculations for better tracking.

Responsive design ensures optimal viewing on all devices from desktop to mobile. Error handling prevents invalid operations like division by zero. Enjoy seamless calculations with visual feedback on every button press.
